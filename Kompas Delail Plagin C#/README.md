##№ Инструкция

- Запускаете плагин
- Нажимаете "Запустить КОМПАС 3D" 
- Создаете в открывшемся КОМПАСе новую деталь
- Заполняете характеристики
- Нажимаете "Создать деталь"
- Закрываете плагин
- Наслаждаетесь
